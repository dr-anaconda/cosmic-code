# cosmic-code

Repositorio maestro de proyecto para **Codigo Cosmico - Evolucion Profunda**.

Este repositorio consolida dos frentes:

- `app/`: base para desarrollo de producto (estructura inicial).
- `docs/`: sistema documental oficial del proyecto.

## Regla de version documental

Cuando exista mas de una version para el mismo tema, siempre prevalece la mas reciente:

- `v1.3` > `v1.2` > `v1.1` > `v1.0`.
