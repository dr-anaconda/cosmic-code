# Estandar de Redaccion de Documentos

Version: 1.0

## Objetivo

Garantizar consistencia de estructura, profundidad y utilidad entre documentos.

## Reglas generales

- Redaccion clara, directa y accionable.
- Secciones orientadas a decisiones, no relleno.
- Coherencia terminologica entre documentos.
- Sin contradicciones con decisiones confirmadas.

## Estructura minima recomendada

1. Proposito del documento
2. Principios y limites
3. Reglas operativas
4. Riesgos y mitigaciones
5. Checklist de validacion
6. Declaracion final

## Reglas de profundidad

- Definir el "que" y el "como".
- Incluir criterios de aceptacion cuando aplique.
- Evitar vaguedades no medibles.

## Regla de versionado

- Todo documento debe tener version y estado.
- Cambios importantes deben actualizar version menor o mayor.
