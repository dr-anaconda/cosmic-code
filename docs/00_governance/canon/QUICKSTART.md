# Canon Quickstart

## Uso en 2 minutos

1. Leer `00_MASTER_CHARTER.md`.
2. Ejecutar Fase 0 con `01_WORKFLOW_PROTOCOL.md`.
3. Abrir ronda con formato de `02_QA_ROUNDS_SPEC.md`.
4. Registrar decisiones en `DECISIONS/decision_log.md`.
5. Redactar con `04_DOCUMENT_AUTHORING_STANDARD.md`.
6. Validar gates en `07_GATES_AND_ENFORCEMENT.md`.
7. Cerrar con `08_DONE_CRITERIA_CHECKLIST.md`.

## Orden recomendado de lectura

- `00_MASTER_CHARTER.md`
- `01_WORKFLOW_PROTOCOL.md`
- `02_QA_ROUNDS_SPEC.md`
- `07_GATES_AND_ENFORCEMENT.md`
