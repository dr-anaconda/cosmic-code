# Especificacion de Rondas de Preguntas

Version: 1.1

## Formato obligatorio de interaccion

Documento: [Nombre del documento]  
Pregunta X: [Texto de la pregunta]  
Opciones:  
1. [Opcion 1]  
2. [Opcion 2]  
3. [Opcion 3]  
4. [Opcion 4]  
(Opcion abierta permitida)

## Reglas de calidad de preguntas

- Deben cerrar una ambiguedad real.
- Deben impactar el documento actual.
- Deben ser claras y accionables.
- Deben evitar jerga innecesaria.

## Reglas de opciones

- 4 opciones concretas y distinguibles.
- Sin opciones redundantes.
- Se permite combinacion de opciones.
- Opcion abierta siempre aceptada.

## Recapitulacion obligatoria

Tras cada respuesta se debe:

1. Confirmar lo elegido.
2. Traducirlo a regla de documento.
3. Indicar impacto practico.
4. Continuar con la siguiente pregunta.

## Cierre de ronda

Una ronda se considera cerrada cuando:

- no quedan dudas criticas abiertas
- hay resumen final de decisiones
- se autoriza paso a redaccion
