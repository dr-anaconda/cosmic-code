# Definition of Done (DoD)

Version: 1.0

## DoD por documento

- [ ] Fase 0 relevante para este documento completada.
- [ ] Ronda de preguntas cerrada y resumida.
- [ ] Decisiones registradas en decision log.
- [ ] Redaccion completada con estandar canonico.
- [ ] Coherencia interna validada.
- [ ] Coherencia externa con documentos previos validada.
- [ ] Gates 0-8 en pass.

## DoD de ciclo completo

- [ ] Todos los documentos objetivo redactados.
- [ ] Sin contradicciones criticas entre documentos.
- [ ] Registro de decisiones actualizado y trazable.
- [ ] Estado de version y fecha en cada documento.
- [ ] Propuesta de siguiente iteracion definida.
