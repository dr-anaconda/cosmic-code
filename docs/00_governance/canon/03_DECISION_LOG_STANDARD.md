# Estandar de Registro de Decisiones

Version: 1.0

## Objetivo

Mantener trazabilidad completa entre preguntas, decisiones y contenido final.

## Estructura de una entrada

- ID: DEC-0001
- Fecha:
- Documento afectado:
- Pregunta origen:
- Opciones elegidas:
- Respuesta abierta (si aplica):
- Regla resultante:
- Impacto esperado:
- Estado: confirmada | revisada | descartada

## Reglas de uso

- Una entrada por decision significativa.
- No borrar decisiones; solo cambiar estado.
- Toda seccion clave de un documento debe mapear al menos una decision.

## Formato recomendado de archivo

`DECISIONS/decision_log.md`
