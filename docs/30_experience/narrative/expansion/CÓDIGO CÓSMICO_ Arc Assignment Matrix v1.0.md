## 4. Reglas de asignacion

- Todo principal tiene 1 arco primario y 1 secundario.
- Todo secundario tiene 1 arco asociado y 1 evento pivote.
- Ningun arco puede concentrar mas del 20% de todo el roster sin DEC de excepcion.

## 5. Reglas de tension dramatica

- Cada arco debe contener al menos 2 facciones en conflicto directo.
- Cada arco AT-05..AT-10 debe incluir costo irreversible visible.
- Cada arco AT-01..AT-03 debe incluir una ruta contemplativa valida.
