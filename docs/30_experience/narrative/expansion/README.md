# Narrative Expansion Pack

Coleccion de contenido narrativo expandido para escalar el canon sin romper coherencia.

## Objetivo

Expandir en volumen (x2) la base de contenido utilizable en runtime narrativo:

- eventos,
- cadenas,
- subfacciones,
- hooks de integracion con game design.

## Relacion con canon

- Base inmutable: `docs/30_experience/narrative/canon/`.
- Este directorio solo agrega capas derivadas, sin retcon de nucleo.
