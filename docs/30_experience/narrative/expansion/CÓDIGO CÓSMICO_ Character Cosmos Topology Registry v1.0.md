## 5. Planetas (300)

Regla:
- 3 planetas por sistema.
- Sufijos: `A`, `B`, `C`.
- Nombre: `<Sistema>-A/B/C`.

Ejemplo de referencia global:
- SYS-001 -> PL-001A Helix-I-A, PL-001B Helix-I-B, PL-001C Helix-I-C.
- SYS-057 -> PL-057A Delta-II-A, PL-057B Delta-II-B, PL-057C Delta-II-C.
- SYS-100 -> PL-100A Null-V-A, PL-100B Null-V-B, PL-100C Null-V-C.

Nota operativa:
- Los 300 nombres quedan determinados de forma completa por esta convención.
