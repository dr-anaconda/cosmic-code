# CÓDIGO CÓSMICO — Glosario Humano

## Objetivo

Traducir términos técnicos del proyecto a lenguaje claro para personas no técnicas.

| Término técnico | En lenguaje humano |
|---|---|
| Implosión | Reinicio del universo con parte del progreso guardado |
| Tick | "Pulso" de actualización del mundo |
| MMR | Puntuación interna de nivel competitivo |
| Balance | Ajuste de reglas para que el juego sea justo e interesante |
| Data Dictionary | Mapa de los datos que guarda el juego |
| Core Loop | Secuencia básica de acciones que repites jugando |
| Asíncrono | No necesitas estar conectado al mismo tiempo que otros |
| Guardrails | Límites para evitar decisiones de diseño dañinas |
| DoR / DoD | Definición de cuándo una tarea está lista para empezar / terminar |
| Pipeline | Cadena ordenada de pasos para producir algo |

## Regla práctica

Si una persona nueva no entiende un término en 10 segundos, hay que explicarlo o reemplazarlo en ese contexto.
